Introduction

Before you start making websites with HTML, you should get familiar with some good practices before making a website.

This guide will show you how to:

  • Create a website
  • Add a title
  • Know the difference between a "Head" and a "Body" of a website
First Steps

In order to use this guide properly it is recommended that you:

  • Know how HTML is written
  • Have a tool to write the code ready
Head vs Body

The head of a website is generally the things you don't see directly on the page such as:

  • The title
  • The connection between the HTML and the CSS
  • The rules for the site (meta tags)
Setting Up Head

To start off with let's tell the internet that this is a HTML document.

To do this we start off the document by typing:

<!DOCTYPE html>

Next to start writing HTML type out:

<html lang="en">

</html>

Now we need to set up the head of a page. Between the HTML opening and closing tag type out:

<head>

</head>

Congrats! You have now setup the head of the page.

Now we need to link our stylesheet to the HTML document. To do this type:

<link rel="stylesheet" href="(location of the stylesheet eg ./styles.css)>

Remember to replace the location with where the stylesheet is located.

Next we add some "meta" tags in order to complete the setup of the head. Type:

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale="1.0">

This is done to tell the browser that it should use the full width of the device and that we are using the "UTF-8" set of characters.

Setting Up Body

To set up the body, type:

<body>

</body>

Congratulations! You are now ready to create your website content!
